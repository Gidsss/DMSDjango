# Legal Document Management System using Enhanced Markov Chain for Steganographic Encryption

A comprehensive Django-based web application to manage legal documents.

## Setup/Installation

1. **Open Terminal/Command Prompt**: 
    - Make sure `python` and `pip` are in your system's environment variables.

2. **Navigate to the Source Code Directory**:
    - Change to the directory where you extracted the source code.
    - Example: `cd C:\Users\Gids\Desktop\django_ams`

5. **Install Dependencies**:
    - Run these commands to install Django and all required modules:
    ```bash
    pip install Django
    pip install -r requirements.txt
    ```

6. **Apply Migrations**:
    - Set up the database tables by applying migrations:
    ```bash
    python manage.py migrate
    ```

7. **Run the Server**:
    - Start the server and access the application:
    ```bash
    python manage.py runserver
    ```
    - In a web browser, navigate to [http://localhost:8000/](http://localhost:8000/) or [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

### Notes
- If any missing modules or libraries occur, install them as prompted.
